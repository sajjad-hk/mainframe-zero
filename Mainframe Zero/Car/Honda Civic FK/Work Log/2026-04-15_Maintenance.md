## ✅ Task List

### Phase 1 — Cooling System Flush
> 🏪 Self service workshop | ⏰ Before end of May | 🌡️ Cold engine only

- [ ] Book self service workshop slot
- [ ] Prepare kit to bring:
  - [ ] Large container ~7L for old coolant
  - [ ] Old towels/rags
  - [ ] Funnel
  - [ ] Kühler Reiniger 300ml
  - [ ] Honda OEM coolant 5L + 1L reserve
- [ ] Watch YouTube: "Honda Civic FK coolant flush" before going
- [ ] **Procedure:**
  - [ ] Confirm engine is cold before starting
  - [ ] Drain old coolant completely
  - [ ] Fill with tap water + Kühler Reiniger 300ml
  - [ ] Run engine to operating temp (~15 min)
  - [ ] Drain completely
  - [ ] Fill with Honda OEM pre-mix 08CLA-G02 (straight from bottle, no water needed)
  - [ ] Run engine, heater on full, burp air from system
  - [ ] Top up with 1L reserve 08CLA-G01 if level drops
  - [ ] Monitor coolant level for a few days afterward
- [ ] Log completion date and mileage
